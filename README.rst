my-mfe plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/anis-meliti/tutor-contrib-my-mfe.git

Usage
-----

::

    tutor plugins enable my-mfe
